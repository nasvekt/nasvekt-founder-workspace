# Contributing to NasVekt Founder OS

Thank you for your interest in contributing! This project aims to provide a robust validation layer for AI-driven business ideas.

## How to Contribute
- **Report Bugs**: Use the GitHub Issue Tracker.
- **Suggest Features**: Open an issue to discuss new pipelines or prompts.
- **Pull Requests**:
  1. Fork the repo.
  2. Create a feature branch.
  3. Submit a PR with a clear description of changes.

## Standards
- Keep prompts modular and evidence-based.
- Ensure any new skills follow the `.skill` format.
